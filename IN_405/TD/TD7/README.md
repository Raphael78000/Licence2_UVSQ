
Une fois "make" écrit dans le terminal : 

Dans un terminal, faire "./aff" : 
Les messages seront affichés dans ce terminal.
La limite du nombre de messages est 10.

Dans un autre terminal, faire "./new" : 
Il suffit d'écrire les messages dans ce terminal 
pour qu'il s'affiche dans l'autre.
ATTENTION : ne pas mettre d'espaces dans les messages.

messages particuliers : 
"quit" -> termine les deux processus
"init" -> ré-initialise la liste de messages aux deux premiers messages
"delete" -> supprime le dernier message de la liste de messages 
            ( ne fais rien s'il y a que 2 messages dans la liste )
